# Homography Demonstration

This repository demonstrates homography on soccer videos.
By running each cell from start to finish, you will be able to create a homographic projection of soccer players and the ball from a video of your choosing.

## Installation

Make sure the Jupyter notebook uses a virtual environment (venv) to not pollute your system packages.

This notebook was tested on Python 3.11.12.

```
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
